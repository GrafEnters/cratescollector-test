ТЗ:

Требования:

Персонаж
- Простой контроллер (капсула или базовый меш)
- Движение WASD
- Камера от третьего лица

Предметы
- 5-6 предметов разбросаны на сцене (кубы с разными цветами — достаточно)
- Данные предметов загружаются из JSON файла: id, название, цвет, stackable (да/нет), максимальный стак
- При приближении к предмету появляется подсказка "Нажмите E"

Инвентарь
- UI-окно с сеткой ячеек (минимум 12 слотов)
- Открытие/закрытие по клавише Tab
- Подобранные предметы отображаются иконкой
- Stackable предметы показывают количество
- Ячейки корректно обновляются при подборе
- Drag & drop предметов между ячейками
- Выброс предмета из инвентаря обратно в мир (ПКМ или перетаскивание за пределы окна)

Подбор
- Игрок подходит к предмету, нажимает E
- Предмет исчезает со сцены, появляется в инвентаре
- Если предмет stackable и уже есть в инвентаре — количество увеличивается

---

Технические требования:

- Unity 6+
- UI: UI Toolkit
- Код на C#
- Никаких сторонних ассетов для логики (визуальные ассеты — можно)

---

Формат сдачи:

1. Ссылка на Git-репозиторий
2. README с инструкцией по запуску
3. Видео 1-2 минуты: демонстрация работы

---

Критерии оценки:

- Работоспособность — всё запускается и работает
- Архитектура — разделение данных, логики и UI
- Чистота кода — читаемость, отсутствие копипасты
- Расширяемость — легко ли добавить новый тип предмета
